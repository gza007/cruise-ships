# Project Title

Cruise Ships

## Getting Started

Fork and/or Clone to your local machine

### Prerequisites

NPM, Node & Jest

### Installing

Install NPM

```
npm install
```

Install Jest

```
npm install -D jest
```

## Running the tests

```
npm run test
```
